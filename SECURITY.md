# Security Policy

## Supported Versions

This project ships rolling releases from `main`. Only the latest tagged
release receives fixes; please upgrade to the latest version before
reporting an issue.

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Instead, report it privately using
[GitHub Security Advisories](https://github.com/teddymalhan/CloudReaper-Alerts/security/advisories/new)
for this repository. Include:

- A description of the vulnerability and its impact
- Steps to reproduce, or a proof of concept
- Any relevant logs, configuration, or AWS resource types involved

You should receive an initial response within a few days. Once a fix is
ready, a new release will be published and the advisory disclosed.
